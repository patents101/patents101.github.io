The present_invention provides in one aspect a method and a system for displaying a VoIP quality to a user  so that the user may determine if a specific_call should be placed in view of the VoIP quality. The present_invention also provides in another aspect a method and a system for non-intrusively_signaling of an incoming VoIP call so that a user can be informed of the incoming_call and yet not be interrupted from his/her current_work.
