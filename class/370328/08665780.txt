The present_invention_relates to a router__computer_program_products and a method in a Universal Mobile Telecommunication System  UMTS . The router_resides in an Internet Protocol  IP  based UMTS Terrestrial Radio Access Network  UTRAN  Transport Network. The UTRAN transport network carries Dedicated Channel  DCH  frames on DCHs between a RNC and at least one Node B  wherein the router_comprises_means for splitting one DCH traffic flow into at least two DCH traffic flows by using an IP multicast protocol.
