The application_relates to wireless_transmission of load_status_information for load_balancing among Home Node Bs  HNB  or Local Area Node Bs  LNB  for which an X2 interface is not available. In particular  an LNB may pretend to be a user_equipment with a specific predefined user_equipment ID value. A neighboring LNB which receives a data_packet with this user_equipment ID value knows that the data_packet contains load_information_monitored  by a neighboring LNB.
